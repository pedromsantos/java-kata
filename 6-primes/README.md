# Prime factors kata

Factorize a positive integer number into its prime factors.

For example:

| **Input** | **Output** |
| --------- | ---------- |
| 2         | `[2]`      |
| 3         | `[3]`      |
| 4         | `[2,2]`    |
| 6         | `[2,3]`    |
| 9         | `[3,3]`    |
| 12        | `[2,2,3]`  |
| 15        | `[3,5]`    |

Source: <http://cyber-dojo.org/>
